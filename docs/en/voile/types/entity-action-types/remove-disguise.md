# Remove Disguise

Removes the player's current [disguise](../bientity-action-types/disguise.md).

Type ID: `voile:remove_disguise`

### Fields

*None.*

### Examples

```json
"entity_action": {
    "type": "voile:remove_disguise"
}
```

This example will remove the player's current disguise.
